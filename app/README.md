# Maratea App Platform

Основное приложение платформы Maratea, доступное на `app.maratea.com`.

## Технологии

- Next.js 14+ (App Router)
- React 18+
- TypeScript
- SSR (Server-Side Rendering)

## Разработка

```bash
pnpm dev
```

Приложение будет доступно на http://localhost:3000

## Структура

Будет создана в Фазе 6.
