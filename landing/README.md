# Maratea Landing Page

Статичный маркетинговый сайт школы, доступный на `maratea.com`.

## Технологии

- Next.js 14+ (Static Export)
- React 18+
- TypeScript

## Разработка

```bash
pnpm dev
```

Сайт будет доступен на http://localhost:3001

## Сборка для продакшена

```bash
pnpm export
```

Сгенерированные статические файлы будут в директории `out/`.

## Структура

Будет создана в Фазе 5.
